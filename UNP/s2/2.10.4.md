# 信号驱动 IO (signal-driven IO)
***

&emsp;&emsp;
信号驱动 IO 的方式是发送 SIGIO 信号来通知某个 fd 是可读或是可写的。
fd 的阻塞等待或者是轮询都让系统来进行托管，而不是在用户态处理。

&emsp;&emsp;
设置信号驱动 IO 可以使用如下代码：

    // 首先设置信号驱动，这样准备好之后会发送 SIGIO 信号
    int flags = fcntl(fd, F_GETFL, 0);
    flags |= O_ASYNC;
    fcntl(fd, F_SETFL, flags);
    // 设置 SIGIO 信号属主
    fcntl(fd, F_SETOWN, getpid());
